JuliaDispatch.Utils.get_new_snapshot — Method
get_new_snapshot(current_snap; data="data/", run="", sleeptime=10, maxsleep=100)

Return snapshot id with ID = current_snap + 1. Will wait for new snapshots as they are being produced. If program waits for more than maxsleep seconds without finding a new snapshot, the program will exit.
