The JuliaDispatch.Select Module

This module contains functionality for locating and extracting patches and data at certain locations within the domain.

JuliaDispatch.Select.box — Method

box(patch::Dict; iv::Union{String, Int}, all::Bool, verbose::Int)

Return 3D-array of data of quantity iv from patch. Shorthand for patch["var"](iv, verbose=verbose, all=all).

Examples

julia> density_box = box(patch, iv="d")
